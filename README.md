EjemploHandlers
===============

En este ejemplo se implementa un ProgressDialog y una ProgressBar utilizando la clase Handler.
  - En el ProgressDialog se utiliza el método View.post().
  - En la ProgressBar se uitiliza el método handleMessage() y el método sendMessage().
El resultado es el mismo pero son dos implementaciones distintas.
